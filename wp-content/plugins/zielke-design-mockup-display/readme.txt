== Description ==
Create scrollable mockups inside monitor graphics that can be added to pages via shortcode.

**Features:**

1. scrollable images displayed in monitor graphics.
1. desktop and mobile displays.
1. light and dark monitor graphics.

== Installation ==
1. Upload "zielke-design-mockup-display" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
